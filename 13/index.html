Wordpress 套件安裝

GA Google Analytics

進入 Wordpress 後台後, 點選「plugins」選單，選擇「Add New」，接著

  • 搜尋「google analytics」
  • 找到「GA Google Analytics」，點擊「install now」
  • 待按鈕變成「activate」後，再次點擊
  • 回到「plugins」選單中，點擊「installed plugins」
  • 找到「GA Google Analytics」
  • 點擊「settings」→「plugin settings」
  • 更新「GA Property ID」
  • 點選 Enable Analytics
  • Save Settings

Yoast SEO

進入 Wordpress 後台後, 點選「plugins」選單，選擇「Add New」，接著

  • 搜尋「Yoast SEO」
  • 找到「Yoast SEO」，點擊「install now」
  • 待按鈕變成「activate」後，再次點擊
  • 之後在編輯貼文時，可在右下角找到「feature image」
  • 編輯介面的最底下則提供臉書標題、描述的編輯器。

Google Search Console

連至 Google Search Console，然後：

  • 登入
  • 點擊「新增內容」
  • 輸入自己的網域，按新增
  • 選擇「其它方法」中的「Google Analytics 分析」，並點「驗證」後「繼續」
  • 加入其它使用者：「設定」→「使用者與資源擁有者」→「新增使用者」

接受 Google Search

在「settings」選單中找到 reading → Search Engine Visibility → uncheck
