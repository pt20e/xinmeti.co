最低要求

  • 完成基本市場調查報告
  • 完成經營策略報告
  • 網站基本設定完成，包含站名、追蹤碼、廣告等
  • 各組員各自完成一頁自我簡介
  • 各組員至少有一篇文章做為主要作者，文章需滿足：
    • 妥善文章外觀與可讀的內容
    • 至少含有一張圖片
    • 妥善的 Facebook 與 Twitter 預覽結果
    • 文章字數 500 字以上
    • 期末評分前瀏覽量達 100 以上。


評分方式

競賽成績分別以個人及分組成績加權計算，並有額外加分機會。成績中各個項目的計分方式如下：

  • 基本要求 - 做到即得分
  • 進階評比 - 各組依各項指標為項目排名，並依排名計分。
    • 第一、二名：5 分
    • 第三～五名：4 分
    • 第六、七名：3 分
  • 額外追加 ( Bonus ) - 特別項目前三名者於總分上額外追加分數。
    • 第一名：3 分
    • 第二名：2 分
    • 第三名：1 分
  • 總評比計分方式：依進階評比各項名次總和決定勝負。
  • 總分計算方式： ( 超過 100 分則以 100 計 )

    總分 =
  ( 分組得分 * 0.4 ) +
  ( 個人得分 * 0.6 ) +
  分組總評比額外追加 +
  個人總評比額外追加 +
  營收排名額外追加

分組評分細節

  • 基本要求 ( 70分 )，其中：
    • 8分 / 基本市場分析報告 ( 第 10 次上課課後作業 )
    • 8分 / 基本規劃報告 ( 第 11 次上課練習 )
    • 20分 / 分組中，每個人至少須有一篇文章為主要作者。
    • 18分 / 妥善設定下列平台並開放讀取權限給老師：
      • 6分 / Google Analytics
      • 6分 / Google Adsense
      • 6分 / Google Search Console
    • 16分 / 妥善設定網站的基本資訊
  • 進階評比 ( 30分 )，其中：
    • 5分 / 整體 Google Adsense 表現 ( 經營冠軍 )
    • 5分 / 網站平均搜尋排名
    • 20分 / 指定的六篇文章的：
      • 5分 / 瀏覽量(總和)
      • 5分 / 停留時間(時間 x 各別瀏覽量/總瀏覽量)
      • 5分 / 跳出率(中位數)
      • 5分 / 臉書按讚數(總和)

個人評分細節

以個人指定的一篇文章做為評量用的專文。

  • 基本要求 ( 80分 )，其中：
    • 5分 / 建立自介頁面，字數不限。
    • 35分 / 專文字數 500 以上，至少含有一張圖，可讀的內容與適當的外觀。
    • 20分 / 專文瀏覽量 100 人次以上。
    • 20分 / 社群預覽良好，有縮圖，妥善的標題與摘要。其中：
      • 10分 / 通過 Facebook Debug 確認
      • 10分 / 通過 Twitter Validator 確認
  • 進階評比 ( 20分 )，其中：
    • 5分 / 瀏覽量
    • 5分 / 停留時間
    • 5分 / 跳出率
    • 5分 / 臉書按讚數
