嵌入內容長寬比

使用 iframe 可輕鬆置入外部網站的互動資源，但有時尺寸是個很令人頭大的問題。有些網站會提供「固定長寬比」的功能，但基本上原理並不複雜，只要理解基本概念，自己也可以實作長寬比固定的 iframe 嵌入。

在 CSS 中，padding 可以用來保留區塊內框的留白。如同 width 或 height 屬性， padding 亦可以使用百分比來設定其大小，這時這個百分比其實是相對於該元素的「寬度」。比方說，

padding-bottom: 50%

代表 padding-bottom 的大小為寬度的一半。

因此，我們可以利用這個特性，將區塊的高度設定為 0 ，再利用 padding-bottom 來保持整體區塊的寬高比：

<div style="width:300px;border:1px solid #999">
  <div style="height:0;padding-bottom:50%">
  </div>
</div>


上例描繪出一個寬高比為 2:1 的矩形。你可以自由的調整 width 的值來實驗其效果。接著，為了讓 iframe 貼合這個矩形的大小，我們讓他相對於最外層元素來進行絕對定址：

<div style="width:300px;border:1px solid #999;position:relative">
  <div style="height:0;padding-bottom:50%">
    <iframe src="https://www.google.com/maps/embed?pb=!1m14!1m12...."
    width="100%" height="100%" frameborder="0"
    style="border:0;position:absolute" allowfullscreen></iframe>
  </div>
</div>


最後，考慮到響應式的效果，將最外層的寬度使用百分比來設定，或搭配 Bootstrap 的 row 等相關屬性來使用。上例程式碼直接將寬度設為 100% 後即得到下例：



Media Query

並非所有時候保留長寬比都是最好的做法。比方說，原本在桌機上適合 landscape (橫向) 形式的地圖，也許在手機上更適合以 portrait (直向) 方式呈現。依照不同媒介來設定不同 CSS 的方式一般稱之為響應適設計，可以透過 CSS 中的 Media Query 來達成。下面為一個 Media Query 的範例：

@media (max-width: 480px) {
  body {
    font-size: 24px;
  }
}

上例會在螢幕寬度不到 480px 時將網頁字體大小設定成 24px 大。搭配 iframe 使用，我們可以在螢幕寬度不到 480px，將 iframe 佔滿整個畫面：

<style type="text/css">
  @media (max-width: 480px) {
    .container {
      height: 100%
    }
    .container div {
      padding-bottom: 0!important;
      height: 100%!important;
    }
  }
</style>
<div class="container" style="width:300px;border:1px solid #999;position:relative">
  <div style="height:0;padding-bottom:50%">
    <iframe src="https://www.google.com/maps/embed?pb=!1m14!1m12...."
    width="100%" height="100%" frameborder="0"
    style="border:0;position:absolute" allowfullscreen></iframe>
  </div>
</div>

要注意的一點是，若這段碼要有效，區塊的所有上層元素 ( 包含 html, body ) 都需要 height: 100% 這樣的 CSS 樣式設定。
